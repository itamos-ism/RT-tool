# RT-tool
Solve the radiative transfer equation for 1D PDR models.

You will need to have the `paramsRT.dat` file and the `RT-tool/` directory in the home directory of 3D-PDR.

Click [here](http://itamos.readthedocs.io/) to visit the manual of the ITAMOS project. You can find the manual of RT-tool in the "Radiative Transfer" section.
